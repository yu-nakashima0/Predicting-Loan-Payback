Categorical encoding:
    Categorical variables can be:
        Nominal: No natural order (e.g. color = red, blue, green)
        Ordinal: Has an inherent order (e.g. size = small, medium, large)
    Type of Data  ->  Best Encoding
        Ordinal (has order)  ->  OrdinalEncoder
        Nominal (no order, few categories)  ->  OneHotEncoder
        Nominal (many categories)  ->  TargetEncoder, BinaryEncoder, or CountEncoder
    
Feature Engineering:
    Wann ist ein Feature sinnvoll?
    -> 1. statistische Zusammenhänge
            Für numerische Ziele: Pearson-Korrelation
            Für kategorische Ziele: ANOVA oder Chi-Quadrat
    -> 2. Modellbasierte Feature-Importance
            Tree-basierte Modelle wie RandomForest oder XGBoost geben Feature-Importances aus.
            Permutation Importance misst den Einfluss eines Features, indem man seine Werte vertauscht.
    -> 3. Verteilungs-Check
            Features mit nahezu konstanter Verteilung (fast alle Werte gleich) bringen wenig Mehrwert.
            Features mit extrem vielen fehlenden Werten können ebenfalls problematisch sein.
